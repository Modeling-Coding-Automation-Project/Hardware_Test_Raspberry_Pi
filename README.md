# リアルタイム制御向けC++制御アルゴリズム Raspberry Pi 環境

## 特徴

C++制御アルゴリズムのメモリ消費、計算時間を Raspberry Pi で検証できる環境です。

いわゆる Processor-in-the-Loop (PIL) の環境です。

## 環境構築

Raspberry Pi にて、本リポジトリをサブモジュールを含めクローンし、ソースコードをビルドして実行してください。
